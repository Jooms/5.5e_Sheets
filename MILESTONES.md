# Development Milestones

The big steps for the project, with the smaller steps that make up each one.

## 1. Finish Updating Core Classes for 2024

Bring every ported class sheet up to the 2024 (5.5e) rules.

- [ ] Barbarian
- [ ] Bard
- [x] Cleric
- [ ] Druid
- [ ] Fighter
- [ ] Monk
- [ ] Paladin
- [ ] Rogue
- [x] Sorcerer
- [ ] Warlock
- [ ] Wizard

## 2. Support Multiclassing

Let one sheet cover a character who has levels in more than one class. This is
the tricky one, and it will probably reshape how a sheet is put together.

- [ ] Design how multiple class panels stack on one sheet
- [ ] Combine features and resources from each class
- [ ] Handle shared spell slots across caster classes
- [ ] Update the picker to build a multiclass combination

## 3. Add More Classes

Go past the current roster with new classes, including material from other books.

- [ ] Add Artificer
- [ ] Support classes from other sourcebooks
